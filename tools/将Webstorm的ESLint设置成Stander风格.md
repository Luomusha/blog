# 将Webstorm的ESLint设置成Stander风格

## 1.打开设置面板=>Code Style=>JavaScript

```
Tab size                           =  2
Indent                              =  2
Continuation indent  =  2
```


## 2.点击右上角的Set from... => PreDefined Style => Javascript Standard Style
